# Install on hardware

## Tasks

- Supported device matrix (machine name, Yocto release, artifact links, install
  method, CI status, known limitations)
- Flash an SD card image
- TEZI install
- UUU install
- Docker/appengine local target

## Board guides

- Raspberry Pi
- Toradex Colibri iMX6ULL
- Variscite iMX8MN
- Variscite iMX8MM
- Toradex Verdin iMX8MM

We prioritize three gold-standard board paths before covering every board: a
low-friction evaluation target (Raspberry Pi), one NXP i.MX target, and one
x86/local Docker target.

Board-specific generated material is in the
[Reference → meta-pantavisor install guides](/reference).
